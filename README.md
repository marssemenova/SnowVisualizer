# Snow Visualizer
###### CSCI4118 project Winter 2026

This project involves implementing an algorithm for the generation of visually and physically
realistic clumps of snow in OpenGL by using the work of C. Zou et al. [1] as a reference. Optimizing time
and space efficiency, as well as attempting to parallelize the algorithm, will be core tasks.
